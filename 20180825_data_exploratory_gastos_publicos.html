Introdução Data Exploratory¶

Nesse estudo vamos realizar uma anlise exploratria a respeito dos dados abertos de Gastos Pblicos de Senadores.

A base de dados foi disponibilizada atraves do Portal da Transparencia do Governo Federal, sendo emitida pelo Ministrio do Planejamento, Desenvolvimento e Gesto.

A partir da limpeza que foi realizada no outro notebook, agora temos input para entender onde esto os principais outliers de gastos financeiros de senadores.

Glossário¶

CEAPS: Cota para Exerccio de Atividade Parlamentar dos Senadores
